About Me


My name is Oscar Lamas, and I came from Mexico City. I was born in Veracruz, Mexico and I'm going into 10th grade. I really enjoy playing video games and coding, and in the future I hope to pursue game design as a career. I came to Summer Discovery with a bit of computer programming experience. I took a semester long web design course, which taught me how to use HTML, CSS, and JavaScript. The final project was to create a website with all three elements. However, I didn't have any experience in Unity.

My Blog

Web Design in UCLA July 11th, 2019

I came to this computer programming course with some experience in web design. Although at first it was a bit slow because I had learned about a lot of this stuff previously, in the end it was really fun. In my school, the web design was really fun, and this was no exception. I personally really enjoy coding and learning new ways to make a website interesting. Overall, the class got a lot better when we switched from Glitch to Visual Studio Code, because then we started to actually code cool stuff.

Unity in UCLA July 11th, 2019

Starting to get into Unity was probably one of the coolest things I've learned with computer programming. Since game design is something I want to pursue in the future, getting Unity was a really cool experience for me. I hadn't ever worked with Unity before, so it was fun to learn how it worked. I had heard about Unity being a very popular game engine alongside Unreal, but never thought to try it. Sadly, the course was very short, but I had a great time in the short classes because I learned a lot more about game design.

Experience at Summer Discovery July 11th, 2019

I was really excited to come to Summer Discovery. Since going into high school, I've been thinking about what college I want to go to. It was important for me and my family that I start to get a little bit of college life experience. I wanted to go to a school that offered computer science or computer engineering. I chose UCLA, because I thought that it would be a really cool experience to go to LA for the first time while living in a college dorm. Overall the activities were really fun and the classes were great. The campus is really nice and the organization was handled very well.
